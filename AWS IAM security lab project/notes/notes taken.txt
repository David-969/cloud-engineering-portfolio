Challenges and Troubleshooting

Region Mismatch

Problem

EC2 instances not visible after login.

Cause

Incorrect AWS region selected.

Resolution

Switched to correct region where infrastructure was deployed.



Permission Denied Behavior

Problem

Developer user unable to create EC2 resources.

Cause

IAM policy restricted modification actions.

Resolution

Verified least privilege policy enforcement was working correctly.

Least Privilege Principle

Implemented limited permissions so users only access resources necessary for their role.

Example:

developer user could view EC2 resources

developer user could NOT create or terminate EC2 instances

Permission Enforcement

Tested AWS permission boundaries by attempting restricted actions.
